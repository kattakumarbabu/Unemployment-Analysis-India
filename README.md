# COVID-19 Unemployment Analysis in India 📊

## Overview
This data science project analyzes the unemployment rate in India, specifically focusing on the massive economic disruption caused by the COVID-19 lockdowns in 2020. 

## Key Insights
* **The COVID Spike:** Pre-pandemic unemployment was around **9.7%**. During the lockdown peak (April-June 2020), it surged to **18.7%**.
* **Worst Affected Regions:** Haryana (27.4%), Tripura (25.0%), and Jharkhand (19.5%) faced the highest average unemployment rates during this period.

## Tech Stack Used
* **Python** (Data processing and logic)
* **Pandas & NumPy** (Data cleaning and manipulation)
* **Plotly & Seaborn** (Interactive data visualization)
* **Streamlit** (Web application deployment)

## How to Run This Project Locally
1. Clone this repository to your local machine.
2. Install the required libraries: `pip install -r requirements.txt`
3. Run the Streamlit dashboard: `streamlit run app.py`
